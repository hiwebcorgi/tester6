## About

Zero-Cost Web Factory는 지속적인 유지비용 없이 웹사이트를 제작하고 관리할 수 있는 혁신적인 솔루션입니다.

### Main Features

* **무료 호스팅**: Vercel의 무료 플랜 활용
* **시각적 편집**: TinaCMS를 통한 직관적인 콘텐츠 관리
* **GitHub 연동**: Git 기반 버전 관리 및 데이터 저장
* **자동 배포**: 콘텐츠 수정 시 자동으로 사이트 재배포
