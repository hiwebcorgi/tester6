# 🛠️ 사용 가능한 스킬 목록 (Skill List)

이 프로젝트에서 사용할 수 있는 스킬들의 목록과 간단한 설명입니다. 각 스킬은 특정 작업을 더 효율적으로 수행하거나 전문가 수준의 결과물을 만들기 위해 준비된 도구 모음입니다.

필요한 작업이 있을 때 저에게 "OO 스킬 써서 해줘"라고 말씀하시거나, 관련 작업을 요청하시면 제가 적절한 스킬을 찾아 제안해 드립니다.

---

## 🎨 디자인 및 아트 (Visual & Art)

| 스킬 이름 | 설명 |
|---|---|
| **algorithmic-art** | **알고리즘 아트 생성**: p5.js를 사용하여 코드로 예술적인 이미지를 만들어냅니다. 생성 예술(Generative Art)이나 독특한 패턴이 필요할 때 사용하세요. |
| **canvas-design** | **시각 디자인 포스터/아트**: 디자인 철학을 바탕으로 아름다운 포스터나 시각 예술 작품을 .png 또는 .pdf로 만들어줍니다. |
| **slack-gif-creator** | **슬랙용 GIF 만들기**: 슬랙에서 사용하기 좋은 재미있는 애니메이션 GIF를 만들어줍니다. (예: 스폰지밥 밈 등) |
| **theme-factory** | **테마 디자인**: 문서, 슬라이드, 웹페이지 등에 적용할 수 있는 컬러 테마와 폰트 스타일을 생성하거나 적용해줍니다. |

## 📝 문서 및 오피스 (Documents & Office)

| 스킬 이름 | 설명 |
|---|---|
| **docx** | **Word 문서 작업**: .docx 문서를 새로 만들거나, 편집하고, 내용을 분석합니다. 서식 유지나 변경 사항 추적도 가능합니다. |
| **pdf** | **PDF 도구**: PDF에서 텍스트나 표를 추출하고, 문서를 합치거나 나누고, 새로운 PDF를 생성합니다. |
| **pptx** | **PPT 프레젠테이션**: 파워포인트 슬라이드를 생성하고 내용을 편집합니다. |
| **xlsx** | **Excel 스프레드시트**: 엑셀 파일을 생성하고, 수식을 다루며, 데이터를 분석하거나 시각화합니다. |
| **doc-coauthoring** | **문서 공동 작성 가이드**: 제안서, 기술 사양서 등 구조화된 문서를 작성할 때 초안 작성부터 검토까지 체계적으로 도와줍니다. |

## 💻 웹 개발 및 코딩 (Web Development)

| 스킬 이름 | 설명 |
|---|---|
| **frontend-design** | **프론트엔드 디자인**: 리액트 컴포넌트나 웹 페이지를 세련되고 현대적인 디자인으로 만들어줍니다. (단순한 AI 느낌이 아닌 프로덕션급 디자인) |
| **web-artifacts-builder** | **웹 아티팩트 빌더**: 복잡한 기능을 가진 인터랙티브한 웹 앱(HTML/React)을 하나의 아티팩트로 만들어줍니다. |
| **web-design-guidelines** | **웹 디자인 감수**: 작성된 UI 코드가 웹 인터페이스 가이드라인을 잘 따르고 있는지, 접근성은 좋은지 점검해줍니다. |
| **webapp-testing** | **웹 앱 테스팅**: Playwright를 사용하여 로컬 웹 애플리케이션을 자동으로 테스트하고 UI 동작을 검증합니다. |
| **vercel-react-best-practices** | **Vercel/React 최적화**: Vercel 엔지니어링 팀의 가이드라인에 맞춰 React/Next.js 코드를 성능 최적화하고 리팩토링합니다. |
| **mcp-builder** | **MCP 서버 빌더**: LLM이 외부 서비스와 통신할 수 있도록 도와주는 MCP(Model Context Protocol) 서버를 만들어줍니다. |

## 🏢 기타 및 유틸리티 (Misc & Utilities)

| 스킬 이름 | 설명 |
|---|---|
| **brand-guidelines** | **브랜드 가이드라인**: Anthropic 공식 브랜드 컬러와 타이포그래피를 아티팩트에 적용해줍니다. (깔끔한 기업용 스타일 필요 시 유용) |
| **internal-comms** | **내부 커뮤니케이션**: 주간 보고, 공지사항, FAQ 등 회사 내부 소통을 위한 글을 전문적인 톤앤매너로 작성해줍니다. |
| **skill-creator** | **스킬 제작**: 새로운 스킬을 만들거나 기존 스킬을 업데이트하는 것을 도와줍니다. |

### 💡 사용 팁
- **어떻게 쓰나요?**: 채팅창에 "PPT로 제안서 초안 만들어줘" 또는 "이 데이터 엑셀로 분석해줘"라고 요청하면 제가 알아서 적절한 스킬(pptx, xlsx 등)을 불러옵니다.
- **궁금한 점**: 특정 스킬에 대해 더 자세히 알고 싶으시면 "frontend-design 스킬은 구체적으로 뭘 할 수 있어?"라고 물어봐 주세요.
