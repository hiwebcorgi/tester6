## 📋 기본 안내 문구 (복사용)

```
안녕하세요, 사장님! 🙌
ZeroPack 유지비 0원 홈페이지 제작 서비스입니다.

빠른 제작을 위해 아래 내용을 준비해 주시면 감사하겠습니다!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 [필수] 준비해 주실 것

1️⃣ 업체명 (상호)
   → 예: 커피향 카페, 김철수 법률사무소

2️⃣ 업종/서비스
   → 예: 카페, 병원, 법률사무소, 쇼핑몰 등

3️⃣ 연락처 정보
   → 전화번호, 이메일, 주소 (지도에 표시됨)

4️⃣ 로고 이미지 (있으시면)
   → PNG 또는 JPG 파일
   → 없으시면 텍스트로 대체 가능해요!

5️⃣ 소개 문구 (한두 줄)
   → 예: "30년 전통 수제 빵집", "친절한 상담, 빠른 해결"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📸 [선택] 있으면 더 좋아요!

✅ 대표 이미지 3~5장
   → 매장 외관, 내부, 제품/서비스 사진
   → 고화질일수록 좋아요 (자동 최적화됩니다)

✅ SNS 또는 기존 홈페이지 주소
   → 참고용으로 활용합니다

✅ 원하시는 색상 테마
   → 예: 파란색 계열, 따뜻한 느낌 등

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💰 도메인 안내

홈페이지 주소(도메인)는 사장님 명의로 직접 구입하시는 게 좋아요!
→ 가비아(gabia.com)에서 연 1~2만원
→ 구입 방법 가이드 드릴게요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

준비되시면 말씀해 주세요!
빠르게 제작 시작하겠습니다 💪
```

## 🎯 상황별 추가 문구

### 가격 문의 시
```
안녕하세요!

ZeroPack은 크몽 결제가로 제작해 드리고,
이후 월 유지비/관리비가 영구적으로 0원입니다!

💡 타 업체 대비 2년이면 최소 50만원 이상 절약되세요.

결제 후 약 3~7일 내 완성되며,
완성 후에는 사장님이 직접 글자/사진을 수정하실 수 있어요!
```

### 수정 방법 문의 시
```
완성된 홈페이지는 사장님이 직접 수정하실 수 있어요!

✅ 글자 수정: 클릭 → 수정 → 저장 (끝!)
✅ 사진 교체: 드래그 앤 드롭
✅ 1분 영상 가이드 제공해 드려요

전문 지식 없이도 쉽게 가능합니다 👍
```

### 도메인 모르시는 분께
```
도메인은 홈페이지 주소예요!
예: mybusiness.com, myshop.kr

가비아(gabia.com)에서 쉽게 구입 가능하시고,
연 1~2만원 정도 비용이 들어요.

구입 방법 가이드 보내드릴까요? 📖
```

### 급하신 분께
```
빠른 제작 원하시면 아래만 보내주세요!

1. 상호명
2. 연락처 (전화번호)
3. 한 줄 소개
4. 로고 (없으면 생략 가능)

기본 템플릿으로 먼저 제작 후,
세부 내용은 나중에 추가/수정 가능해요!
```
