# [STRATEGY] 크몽 소상공인 타겟: 고회전·고효율 템플릿 TOP 10

## 1. 인테리어/시공 전문 업체 (Trust Builder)
- 핵심: 고화질 포트폴리오 갤러리 + 무료 견적 상담 폼.
- 이유: 단가가 높은 업종이라 신뢰도가 생명입니다. 유지비 0원 모델은 마케팅비가 절실한 인테리어 사장님들께 강력한 유인책이 됩니다.

## 2. 전문직 (변호사·세무사·노무사) (Professional Edge)
- 핵심: 권위 있는 프로필 + 전문 분야 소개 + FAQ + 비공개 상담 예약.
- 이유: 신뢰감 있는 깔끔한 디자인만으로도 수백만 원을 지불하는 시장입니다. 복잡한 기능보다 '격조 있는 정적 페이지'의 정석을 보여주세요.

## 3. 요식업/대형 카페 (Brand Experience)
- 핵심: 감성적인 메뉴판 + 인스타그램 연동 + 매장 오시는 길(네이베 지도).
- 이유: 인스타그램 링크만으로는 부족함을 느끼는 사장님들이 많습니다. "매달 나가는 돈 없이 우리 가게만의 브랜딩 페이지를 가지세요"라는 카피가 잘 먹힙니다.

## 4. 프라이빗 운동 시설 (필라테스·PT·요가) (Visual Attraction)
- 핵심: 강사 프로필 + 시설 전경 + 무료 체험 신청 섹션.
- 이유: 이미지 위주의 업종이라 Next.js의 빠른 로딩 속도가 강력한 무기가 됩니다. 체험 신청 폼은 바로 매출로 연결되는 핵심 기능입니다.

## 5. 학원/교육 서비스 (Information Center)
- 핵심: 강사진 이력 + 커리큘럼 소개 + 합격 수기 + 입학 상담 폼.
- 이유: 설명해야 할 정보량이 많습니다. 사장님이 직접 커리큘럼이나 시간표를 수정할 수 있는 GUI 기능이 가장 빛을 발하는 업종입니다.

## 6. 뷰티/에스테틱 (Review Focus)
- 핵심: 비포/애프터 갤러리 + 시술 가격표 + 네이버 예약 버튼 연동.
- 이유: 사진 업데이트가 잦습니다. 제작자에게 매번 요청하기 힘든 영역을 사장님이 직접 관리하게 해주면 만족도가 매우 높습니다.

## 7. 1인 프리랜서/강사 포트폴리오 (Identity)
- 핵심: 경력 로드맵 + 작업물 리스트 + 협업 문의 창구.
- 이유: 고정 수입이 불규칙한 프리랜서에게 '평생 유지비 0원'은 거절할 수 없는 제안입니다.

## 8. 제조업/B2B 중소기업 (Corporate Static)
- 핵심: 기업 비전 + 주요 생산 품목 + 인증서/특허 현황 + 제휴 문의.
- 이유: 기술력은 좋으나 웹사이트가 10년 전 수준인 곳이 많습니다. 세련된 '수출용/입찰용' 사이트 수요가 꾸준합니다.

## 9. 원데이 클래스/공방 (Experience Marketing)
- 핵심: 수업 과정 영상/사진 + 날짜별 일정 안내 + 수강생 후기.
- 이유: 트렌드에 민감한 업종입니다. 감각적인 레이아웃에 '빠른 속도'를 강조하면 전환율이 높습니다.

## 10. 이벤트/프로모션 랜딩페이지 (Conversion King)
- 핵심: 카운트다운 타이머 + 강력한 CTA 버튼 + 한정 혜택 강조.
- 이유: 특정 시즌이나 광고 집행 시 단기적으로 필요한 페이지입니다. "광고비도 비싼데 서버비까지 내지 마세요"라는 논리가 강력합니다.
