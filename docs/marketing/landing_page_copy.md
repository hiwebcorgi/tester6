# ZeroPack Homepage Content Strategy
Based on: `docs/marketing/kmong_ment.md`

## 1. Hero Section
- **Headline**: 사장님, 언제까지 홈페이지 '월세'를 내실 겁니까?
- **Subheadline**: 초기 제작비 1회면 끝! 평생 유지비 0원 + 직접 수정 GUI 홈페이지.
- **CTA Button**: 지금 당장 탈출하기
- **Social Proof**: "Vercel/GitHub 글로벌 인프라 기반"

## 2. Problem Section (돈이 줄줄 새는 이유)
- **Title**: 사장님의 돈이 줄줄 새고 있는 이유
- **Description**: 월 2~4만 원의 호스팅비? 사실 서버리스 기술로는 $0입니다. 글자 하나 고치는 데 비용 청구? 사장님이 직접 1초 만에 할 수 있는 일입니다.
- **Key Points**:
  - '의도된 불편'과 '낭비'를 기술적으로 제거.
  - 유지비는 '어쩔 수 없는 지출'이 아닌 '기술적 한계로 인한 손실'.

## 3. Comparison Section (압도적 비교)
- **Table Data**:
  - 타사: 월 유지비 30,000원+, 수정 비용 건당 발생 -> 3년 108만원+ 손실.
  - ZeroPack: 월 유지비 0원, 수정 비용 0원 -> 3년 0원.

## 4. Features Section (왜 10배 더 혁신적인가?)
- **Feature 1**: "코딩 몰라도 됩니다" (No-Code GUI)
  - 워드프로세서처럼 화면 위에서 글자와 이미지를 직접 클릭해 바꾸세요.
- **Feature 2**: "서버비가 물리적으로 0원입니다" (Serverless)
  - 전 세계 1위 인프라(Vercel/GitHub)를 활용하여 트래픽이 몰려도 비용 0원.
- **Feature 3**: "구글/네이버 검색 상단 노출" (SEO)
  - 일반 사이트보다 5배 빠른 속도, 반응형 기본 지원.

## 5. Target Audience Section (이런 분들은 '탈출'하세요)
- **Title**: 이런 분들은 지금 바로 '탈출'하세요
- **Targets**:
  - 매달 나가는 홈페이지 호스팅비가 '생돈'처럼 느껴지시는 분.
  - 간단한 공지사항 하나 올리는 데도 제작자 눈치를 봐야 했던 분.
  - 보안, DB 관리 등 머리 아픈 기술 용어에서 해방되고 싶은 분.
  - 한 번 제작으로 10년 이상 추가 지출 없이 쓰고 싶은 실속파 사장님.

## 6. Tech Stack Section (압도적 성능의 비밀)
- **Title**: 압도적 성능의 비밀: Next.js + Serverless
- **Description**: Facebook, Netflix가 사용하는 최신 기술 스택을 그대로 적용했습니다.
- **Techs**: Next.js (속도), Vercel (인프라), TinaCMS (편집), GitHub (저장소).

## 7. Service Process Section
- **Step 1**: 주문 접수 (업종 및 핵심 키워드 파악)
- **Step 2**: 엔진 배포 (0원 유지비 인프라 즉시 구축)
- **Step 3**: GUI 활성화 (사장님 전용 시각적 편집 도구 세팅)
- **Step 4**: 소유권 이전 (모든 권한 100% 양도)

## 8. Pricing Section (투명한 가격 정책)
- **Standard (15만)**: 유지비 0원 홍보형 랜딩페이지.
- **Deluxe (35만)**: *Best Choice* 아임웹처럼 직접 고치는 홈페이지 (다중 페이지).
- **Premium (55만)**: 비즈니스 자동화 솔루션.

## FAQ Section
- **Q1**: 정말 유지비가 0원인가요?
  - A: 네. GitHub와 Vercel의 Free Tier 정책을 활용하여 상업적 이용에도 평생 무료 서버를 구축해 드립니다.
- **Q2**: 수정은 어렵지 않나요?
  - A: 블로그 글쓰기보다 쉽습니다. 제공되는 관리자 화면에서 텍스트를 클릭하고 수정하면 즉시 반영됩니다.
- **Q3**: 도메인은 어떻게 하나요?
  - A: 도메인 구입 비용(약 2만원/년)은 유일하게 발생하는 실비이며, 가비아/후이즈 등 등록 대행사에 직접 납부하시면 됩니다.

- **Title**: 리스크 제로 보장
- **Description**: "제작 후 평생 유지비 0원이 아님이 확인될 경우, 이유 불문 제작비의 200%를 환불해 드립니다."
- **Button**: 선착순 도입 문의하기
