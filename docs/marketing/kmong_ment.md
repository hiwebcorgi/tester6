# [크몽 상세페이지] 사장님, 언제까지 아임웹·식스샵에 '매달 월세'를 내실 겁니까?
# 초기 제작비 1회면 끝! 평생 유지비 0원 + 직접 수정 GUI 홈페이지

## 1. 사장님의 돈이 줄줄 새고 있는 이유 (First Principles Analysis)
대부분의 홈페이지 제작 업체는 사장님을 '플랫폼'에 가속시킵니다.
- 월 2~4만 원의 호스팅비? 사실 서버리스 기술로는 $0입니다.
- 글자 하나 고치는 데 비용 청구? 사장님이 직접 1초 만에 할 수 있는 일입니다.
- 큰 기업은 전담 관리자가 필요하지만, 소상공인에게 월 1만 원의 유지비는 낭비입니다. 10년이면 120만 원입니다.
- 본 서비스는 이러한 '의도된 불편'과 '낭비'를 기술적으로 완전히 제거했습니다.

## 2. 압도적 비교: 타사 vs 본 서비스 (Idiot Index 분석)
유지비는 '어쩔 수 없는 지출'이 아니라 '기술적 한계로 인한 손실'입니다.

[타사 솔루션 (국내 호스팅/빌더사)]
- 월 유지비: 30,000원 x 36개월 = 1,080,000원
- 트래픽 제한: 일 500MB~1GB 수준 (초과 시 차단)
- 수정 비용: 별도 발생 (건당 3~5만 원)
- 결과: 쓰면 쓸수록 손해 보는 구조

[본 서비스 (Next.js + Serverless)]
- 월 유지비: 0원 x 36개월 = 0원 (영구 무료)
- 트래픽 제한: **월 100GB 기본 제공 (국내 최저가 호스팅 대비 약 10배)**, GitHub 저장소 무제한 활용
- 수정 비용: 0원 (사장님이 직접 GUI로 수정)
- 결과: 초기 비용 1회로 평생 고정비 삭제

## 3. 핵심 기능: 왜 10배 더 혁신적인가?

① "디자인은 템플릿으로, 내용은 내 맘대로" - 실속형 제작 방식
- 미리 만들어진 검증된 **고퀄리티 템플릿** 중 하나를 고르시면 그대로 배포됩니다.
- 배포 후에는 **안에 들어가는 글자와 이미지**를 사장님이 원하시는 대로 언제든 자유롭게 수정하시면 됩니다.
- 디자인 고민하느라 시간 낭비하지 마세요. 내용은 100% 커스텀 가능합니다.

② "기능은 타협하지 않습니다" - 문의폼, 게시판, FAQ 완벽 지원
- 단순 소개 페이지만 만드는 것이 아닙니다.
- 고객의 문의를 받는 **문의하기(Contact Form)**
- 공지사항을 직접 올리는 **게시판(Board)**
- 자주 묻는 질문을 정리하는 **FAQ**
- 이 모든 기능이 서버비 0원으로 작동합니다.

③ "물리적 한계를 넘었습니다" - 10배 더 넉넉한 용량
- 국내 호스팅 업체들이 '용량 장사'를 할 때, 저희는 글로벌 표준(Vercel/GitHub)을 따릅니다.
- 트래픽 걱정 없이 고화질 이미지를 마음껏 올리셔도 됩니다.

④ "구글/네이버 검색 상단 노출" - 초고속 Next.js 엔진
- 느린 사이트는 고객을 놓칩니다. 일반 사이트보다 5배 빠른 속도로 SEO 점수를 압도합니다.
- 모바일, 태블릿, PC 어디서든 완벽하게 작동하는 반응형은 기본입니다.

## 4. 서비스 프로세스 (The Automation Algorithm)
1. 주문 접수: 업종 및 핵심 키워드 파악
2. 엔진 배포: 0원 유지비 인프라 즉시 구축 (Vercel/GitHub)
3. GUI 활성화: 사장님 전용 시각적 편집 도구(TinaCMS) 세팅
4. 소유권 이전: 사장님 계정으로 모든 권한 100% 양도 (제작자 개입 종료)

## 5. 이런 분들은 지금 바로 '탈출'하세요 (Target Audience)
대규모 IT 인프라가 필요한 '엔터프라이즈'보다는, **실속과 효율이 중요한 '합리적 리더'**를 위해 설계되었습니다.

- **합리적 경영을 추구하는 중소·중견기업/스타트업**
  - "홈페이지는 명함이자 브랜딩이다"라고 생각하지만, 매달 나가는 고정비는 아까우신 대표님.
- **신뢰도가 생명인 전문직 (병의원, 로펌, 세무)**
  - 깔끔한 디자인과 빠른 속도로 고객에게 신뢰를 줘야 하지만, 복잡한 관리는 싫으신 원장님.
- **퍼스널 브랜딩이 필요한 1인 기업/크리에이터**
  - 나만의 포트폴리오나 홍보 페이지가 필요하지만, 큰 비용 투자는 부담스러운 전문가.
- **기존 호스팅 업체의 '월세' 관행에 지친 사장님**
  - "내 홈페이지인데 왜 매달 돈을 내야 하지?"라는 의문을 가지셨던 분.
