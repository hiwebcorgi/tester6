# [MANUAL] 1분 만에 끝내는 사장님 직접 수정 가이드
# "코딩은 저희가 했습니다. 사장님은 '클릭'만 하세요."

## 1. 관리자 모드 접속 (10초)
- 사장님의 홈페이지 주소 뒤에 [/admin]을 입력하고 엔터를 누르세요.
- 예: [www.사장님가게.com/admin](https://www.사장님가게.com/admin)
- 부여받은 아이디와 비밀번호로 로그인하면 즉시 '편집 모드'가 활성화됩니다.

## 2. 화면 보면서 바로 고치기 (30초)
- [시각적 편집]: 화면 위에 마우스를 올리면 수정 가능한 영역에 '파란색 박스'가 생깁니다.
- [텍스트 수정]: 바꾸고 싶은 글자를 클릭하세요. 왼쪽 입력창에 내용을 타이핑하면 화면에 즉시 반영됩니다.
- [이미지 교체]: 사진을 클릭하고 사장님의 컴퓨터/폰에 있는 파일을 업로드하세요. 즉시 사진이 교체됩니다.

## 3. 저장 및 자동 배포 (20초)
- 수정이 끝났다면 상단의 [Save(저장)] 버튼을 누르세요.
- [자동 업데이트]: 사장님이 저장을 누르는 순간, 전 세계 서버가 사장님의 수정을 감지하고 사이트를 다시 만듭니다.
- 약 1분 후, 실제 사이트에 접속해 보면 수정된 내용이 완벽하게 반영되어 있습니다.

## 4. 사장님이 직접 할 수 있는 것들
- 메인 홍보 문구 및 공지사항 수정
- 메뉴판 가격 및 신메뉴 사진 업데이트
- 이벤트 배너 교체 및 팝업 설정
- 고객 상담 카카오톡 링크 변경

## 5. 절대 깨지지 않는 시스템 (Safety First)
- 레이아웃이나 복잡한 코드는 저희가 잠가두었습니다.
- 사장님이 글자를 아무리 길게 써도 디자인이 망가지지 않도록 '방어 설계'가 되어 있으니 안심하고 수정하세요.
- 수정하다 막히면? '저장'을 누르지 않고 창을 닫으면 이전 상태로 유지됩니다.
