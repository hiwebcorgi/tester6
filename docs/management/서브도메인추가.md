# Cloudflare 서브도메인 (zeropack.corg.app) 설정 가이드

Cloudflare에서 구매한 도메인에 서브도메인을 추가하는 과정은 DNS 레코드를 등록하는 것만으로 간단히 완료됩니다.

---

## 1. DNS 레코드 추가 단계
1. **Cloudflare 대시보드** 접속 후 `corg.app` 도메인을 선택합니다.
2. 왼쪽 사이드바 메뉴에서 **DNS > 레코드(Records)**로 이동합니다.
3. **레코드 추가(Add record)** 버튼을 클릭합니다.

---

## 2. 연결 대상에 따른 입력값 (택 1)

사용 목적에 맞춰 아래 두 가지 방법 중 하나를 입력하세요.

### Case A: 개인 서버(고정 IP)에 연결할 경우 (A 레코드)
* **Type:** `A`
* **Name:** `zeropack`
* **IPv4 address:** `사용자의 서버 IP 주소` (예: 123.123.123.123)
* **Proxy status:** `Proxied (주황색 구름)` 권장

### Case B: 외부 호스팅(Vercel, GitHub Pages 등)에 연결할 경우 (CNAME 레코드)
* **Type:** `CNAME`
* **Name:** `zeropack`
* **Target:** `호스팅 업체에서 제공한 주소` (예: cname.vercel-dns.com)
* **Proxy status:** `Proxied (주황색 구름)`

---

## 3. 필수 체크 사항 (SSL/HTTPS)
`.app` 도메인은 보안 정책상 **HTTPS 접속이 의무화**되어 있습니다.

1. **SSL/TLS 설정:** Cloudflare 메뉴의 `SSL/TLS > 개요`에서 모드를 **전체(Full)** 또는 **가변(Flexible)**으로 설정하세요.
2. **HSTS 활성화:** 브라우저에서 안전하게 접속되도록 SSL 설정이 완료되었는지 확인이 필요합니다.

---

## 4. 설정 확인 방법
설정 후 터미널(또는 CMD)에서 아래 명령어를 입력하여 정상적으로 연결되었는지 확인하세요.

```bash
nslookup zeropack.corg.app
