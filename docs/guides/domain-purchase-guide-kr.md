## 🔵 Step 1: 후이즈 접속 & 검색 (30초)

1. **[whois.co.kr](https://whois.co.kr)** 접속
2. 메인 화면 검색창에 **원하는 도메인** 입력
   - 예: `mybusiness` 또는 `myshop`
3. **검색** 버튼 클릭

---

## 🟢 Step 2: 도메인 선택 (30초)

1. 검색 결과에서 사용 가능한 도메인 확인
   - ✅ **등록 가능** → 선택 가능
   - ❌ **등록 불가** → 다른 이름으로 재검색
2. 원하는 도메인 **체크** 후 **장바구니 담기**

> 💡 **추천**: 비즈니스라면 `.co.kr`이 가장 저렴하고 전문적!

---

## 🟡 Step 3: 회원가입 & 결제 (2분)

### 3-1. 회원가입 (처음인 경우)
1. **간편 회원가입** 클릭
2. 휴대전화 본인 인증
3. 기본 정보 입력

### 3-2. 등록자 정보 입력
```
✅ 이름 (실명)
✅ 휴대전화 번호
✅ 이메일 주소 (중요! 인증메일 수신)
✅ 주소 (도메인 등록 정보로 사용)
```

### 3-3. 결제
- 카드결제 / 계좌이체 / 네이버페이

### 3-4. 등록 완료!
- 이메일로 **도메인 등록 완료** 메일 수신

---

## 🔧 Step 4: Vercel 도메인 연결 (1분 30초)

### 4-1. Vercel에서 도메인 추가

1. **[vercel.com](https://vercel.com)** 로그인
2. 프로젝트 선택 → **Settings** → **Domains**
3. 구입한 도메인 입력 (예: `mybusiness.co.kr`) → **Add**
4. Vercel이 보여주는 **DNS 레코드 메모**

### 4-2. 후이즈에서 DNS 설정

1. **[whois.co.kr](https://whois.co.kr)** 로그인
2. 상단 메뉴 **My도메인** 클릭
3. 해당 도메인 **관리** 버튼 클릭
4. **DNS 관리** 또는 **네임서버 관리** 클릭

### 4-3. DNS 레코드 추가

#### A 레코드 추가 (메인 도메인용)
| 항목 | 값 |
|------|---|
| 타입 | **A** |
| 호스트 | **@** (또는 빈칸) |
| 값/IP주소 | **76.76.21.21** |
| TTL | 3600 (기본값) |

→ **저장** 클릭

#### CNAME 레코드 추가 (www용)
| 항목 | 값 |
|------|---|
| 타입 | **CNAME** |
| 호스트 | **www** |
| 값/도메인 | **cname.vercel-dns.com** |
| TTL | 3600 (기본값) |

→ **저장** 클릭

## ✅ Step 5: 연결 확인 (5~30분 대기)

1. Vercel **Settings** → **Domains**로 돌아가기
2. 도메인 옆에 **✓ Valid Configuration** 표시 확인
3. 브라우저에서 `https://mybusiness.co.kr` 접속 테스트

## 📞 자주 묻는 질문

### Q: .kr과 .co.kr 중 뭘 선택해야 하나요?
**A:** 
- **.co.kr** → 일반 비즈니스, 스타트업 ⭐추천 (저렴)
- **.kr** → 공식적인 느낌, 대기업 이미지

### Q: 도메인 연결 후 바로 안 되는데요?
**A:** DNS 전파에 5~30분(최대 48시간) 걸릴 수 있어요. 30분 후 다시 확인해주세요.

### Q: 다른 업체(가비아, 카페24)에서 샀어도 되나요?
**A:** 네! DNS 레코드 설정 방법은 거의 동일합니다.

### Q: 국제 도메인(.com)은요?
**A:** [Cloudflare 가이드](./domain-purchase-guide.md)를 참고하세요. 더 저렴합니다!
